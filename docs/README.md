# Finch Documentation

Welcome to the Finch documentation.

## What is Finch?

Finch is a lightweight, keyboard-first terminal workspace platform written in Rust and built on Ratatui. It is designed for developers, DevOps engineers, SREs, platform engineers, network engineers, security professionals, and homelab users.
